WorldRoots v0.1

RPG old-school minimaliste en JavaScript

WorldRoots est un RPG tour par tour à l'ancienne, dans l'héritage des tous premiers Final Fantasy.

À l'origine, il s'agit d'un projet pour mettre en application JavaScript dans le cadre d'une formation. Toutefois, j'aime à penser qu'il puisse un jour faire le bonheur de quelque aventurier vidéoludique en quête de nostalgie rôlistique.

Je vous souhaite un bon moment par ici !

v0.1 : Release initiale.

  • Le mode histoire est indisponible.
  • Le mode arcade présente des fonctionnalités limitées.
